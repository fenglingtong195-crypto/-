#思维流动 

#这是一个用于记录与呈现“设计思维的流动性过程”的长期项目

#本仓库不以完成度和结果为导向，而是关注：-思考如何发生   -判断如何形成   -结构如何被不断修正

#这个仓库包含什么？

#内容可能来自但不限于：-空间设计的概念性思考   -突发灵感的结构化拆解   -来自电影、音乐、影像等诸多媒介的跨领域联想，它们并非“方案展示”，而是“思维推演、思维流动、思维叠变的痕迹”

#它们并非“方案展示”，而是”思维推演的痕迹“

#如何阅读这个仓库？

-整体思维方向与方法说明 
-文件/文件夹：不同场景下的阶段性记录  
-提交记录（commit):思维随时间变化的轨迹

#关于使用与参考：这里展示的是**思考方式**，而非可直接复用的设计成果，如果你对某种思维路径感兴趣，欢迎交流与讨论。
